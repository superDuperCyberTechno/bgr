# Better Games Repository
In essence, the *BGR* is a collection of tabletop game rules, all playable with regular game supplies; cards, dice, tokens...

More broadly, it is a collaborative effort to devise, test, discuss, modify and archive *better* tabletop games. What the hell is that supposed to mean? - Well, many books have been written to document the myriad of generic tabletop games that exist out there. A standard 52-card deck is for all intents and purposes an infinite source of entertainment.
